PromptScan Playground

Paste OpenAI / Anthropic / LangChain / litellm code and see the token count and dollar cost of each prompt — computed entirely in your browser, nothing uploaded. This is a single-file demo of the PromptScan CLI; the CLI also scans whole repos, resolves prompts across files, and gates cost regressions in CI.
